# Fun With Triangles

 Fun with Triangle is a web app which can be used to calculate the area and  hypotenous of the triangle, and also quiz can be played . 

## Tech Used

1. Vanilla js
2. CSS
3. HTML

- Check the live app here: https://trianglegenius.netlify.app/
